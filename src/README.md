# RecipeDesk

これは、日々の料理レシピを管理するために作られた、シンプルなデスクトップアプリケーションです。

## 概要

RecipeDeskは、個人のレシピコレクションを簡単に追加、編集、閲覧、削除できる直感的なインターフェースを提供します。Tkinterを使用して構築されており、ローカルのSQLiteデータベースにデータを保存するため、外部のWebサーバーやアカウントは不要です。

## 主な機能

- **レシピ一覧表示**: 保存されているレシピを画像付きのカード形式で表示します。
- **レシピ詳細表示**: 材料、手順、画像を一つの画面で確認できます。
- **レシピ登録**: 新しいレシピをタイトル、画像、材料、手順と共に登録します。
- **レシピ編集**: 既存のレシピ情報を更新します。
- **レシピ削除**: 不要になったレシピを削除します。

## 必要なもの

- Python 3.x

## 環境構築

1.  このリポジトリをクローンまたはダウンロードします。
2.  プロジェクトのルートディレクトリに移動します。
3.  必要なライブラリをインストールします。

    ```bash
    pip install -r src/requirement.txt
    ```

    `requirement.txt` には以下のライブラリが含まれています:
    - `Pillow`: レシピの画像処理に使用します。

## 実行方法

アプリケーションを起動するには、以下のコマンドを実行してください。

```bash
python src/app.py
```

初回起動時に、`src/data/` ディレクトリ内に `recipe.db` という名前のデータベースファイルが自動的に作成されます。
