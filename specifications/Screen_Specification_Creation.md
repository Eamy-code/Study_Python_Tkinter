# RecipeDesk --- Screen Specification（画面仕様書）

Version: 1.0\
Author: 前原 裕輝\
Updated: 2025-12-03

------------------------------------------------------------------------

# 1. 概要

RecipeDesk
は、レシピの登録・一覧表示・詳細閲覧を行うシンプルなデスクトップアプリである。\
Tkinter + MVC構成を採用し、CSV を簡易データベースとして管理する。

本書では、以下 3 画面の UI 仕様を定義する：

-   一覧画面（ListView）
-   詳細画面（DetailView / Read）
-   登録画面（CreateView）
-   共通ヘッダー（全画面共通）

------------------------------------------------------------------------

# 2. 共通ヘッダー仕様（Common Header）

    ┌──────────────────────────────────────────┐
    │ RecipeDesk                              一覧へ  登録へ │
    └──────────────────────────────────────────┘

## 2.1 概要

全画面共通で表示されるヘッダー（ナビゲーションバー）。\
左側にアプリタイトル、右側に画面遷移ボタンを配置する。

## 2.2 要素

  項目         内容
  ------------ --------------------------------
  タイトル     「RecipeDesk」太字／左寄せ
  ボタン       「一覧へ」「登録へ」右寄せ
  レイアウト   Grid または pack（左右に配置）

## 2.3 仕様

-   どの画面からでも遷移可能\
-   別ウィンドウを開かず、画面切替を行う\
-   高さは固定とする

------------------------------------------------------------------------

# 3. 一覧画面（ListView）

## 3.1 画面レイアウト

    ┌──────────────────────────────────────────┐
    │ RecipeDesk                              一覧へ  登録へ │
    ├──────────────────────────────────────────┤
    │ [画像1]        [画像2]                                   │
    │ タイトル1       タイトル2                                │
    │----------------------------------------------------------│
    │ [画像3]        [画像4]                                   │
    │ タイトル3       タイトル4                                │
    │----------------------------------------------------------│
    │ ...                                                      │
    └──────────────────────────────────────────┘

## 3.2 要素

  項目           内容
  -------------- ---------------------------------------
  レシピカード   画像 + タイトル
  画像サイズ     100〜200px のサムネイル
  配置           2列グリッド × N行
  スクロール     必要に応じて実装
  遷移           画像 or タイトルクリック → 詳細画面へ

## 3.3 挙動

-   起動時に CSV を読み込み、全レシピを表示する
-   カードクリックで該当 ID を詳細画面に渡す
-   レシピ数に応じて行が増える

------------------------------------------------------------------------

# 4. 詳細画面（DetailView / Read）

## 4.1 画面レイアウト

    ┌──────────────────────────────────────────────┐
    │ RecipeDesk                        一覧へ  登録へ │
    ├──────────────────────────────────────────────┤
    │              レシピタイトル                              │
    ├──────────────────────────────────────────────┤
    │ [画像（左上固定サイズ）]     |   材料1〜9（右上）            │
    │                                |   空欄は空で表示            │
    ├──────────────────────────────────────────────┤
    │ 作り方（Step 1〜6）横幅いっぱい                               │
    ├──────────────────────────────────────────────┤
    │ [戻る]                                           [次へ] │
    └──────────────────────────────────────────────┘

## 4.2 要素

  項目       内容
  ---------- -------------------------
  タイトル   画面上部中央に太字表示
  画像       左側に300×200程度で表示
  材料       右側に縦並び 1〜9枠
  手順       下段に 1〜6 表示
  ボタン     下部に「戻る」「次へ」

## 4.3 挙動

-   「戻る」→ 前のレシピ（最初なら最後へループ）
-   「次へ」→ 次のレシピ（最後なら最初へループ）
-   「一覧へ」ボタン → 一覧画面へ戻る

------------------------------------------------------------------------

# 5. 登録画面（CreateView）

## 5.1 画面レイアウト（縦配置）

    ┌──────────────────────────────────────────┐
    │ RecipeDesk                              一覧へ  登録へ │
    ├──────────────────────────────────────────┤
    │ タイトル入力欄                                          │
    │--------------------------------------------------------│
    │ 材料（Ingredient 1〜9）                                 │
    │ 1. [________]                                           │
    │ ...                                                     │
    │ 9. [________]                                           │
    │--------------------------------------------------------│
    │ 作り方（Step 1〜6）                                     │
    │ 1. [__________________________________________]         │
    │ ...                                                     │
    │ 6. [__________________________________________]         │
    │--------------------------------------------------------│
    │ 画像アップロード（Explorer + ドラッグ＆ドロップ）             │
    │ [プレビュー表示]                                         │
    │ [ファイルを選択]ボタン                                  │
    │--------------------------------------------------------│
    │                    [登録（Submit）]                     │
    └──────────────────────────────────────────┘

## 5.2 要素

  項目           内容
  -------------- -----------------------------------
  タイトル       Entry
  材料1〜9       Entry（空欄可）
  ステップ1〜6   Entry or Text（空欄可）
  画像           選択ボタン + D&D対応 + プレビュー
  Submit         CSV に保存し一覧へ戻る

## 5.3 挙動

-   SubmitでCSVへ追加行として保存
-   画像は `/images` にコピーしてパス保存
-   入力不足があれば空欄で補完

# 7. 開発メモ（補足）

-   画像は Pillow を使用してリサイズ・表示
-   CSV は DictReader / DictWriter を利用
-   各 View は Frame として生成し、Controller が切り替える
-   材料・手順の数が不足する場合は空欄で埋める
